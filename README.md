# Expense/Income Tracker

A simple expense and income tracker built using HTML, CSS, and JavaScript that works in your browser.

🔗 Live Demo: **https://expinc-tracker.netlify.app**

---

## Overview

This project is a simple personal finance tracker that allows you to record income and expenses, view your balance, and manage transactions directly in your browser.

It is designed to be simple, fast, and easy to use without requiring an account or backend.

All data is stored locally using the browser's localStorage.

---

## Features

- Add income and expense transactions  
- Edit existing transactions  
- Delete individual transactions  
- Clear all data  
- Automatic balance calculation  
- Monthly summary (income, expenses, net)  
- Filter by type (income/expense)  
- Filter by month  
- Search transactions  
- Export data to CSV  
- Clean layout

---

## Data Storage

This project uses **localStorage**, meaning:

- Data is saved in your browser
- Data does **not** sync across devices
- Data **may be lost** in incognito/private browsing sessions

- JavaScript (Vanilla)
